<h1 align="center">Ghost eShop Website</h1>

This repo is the main website of Ghost eShop project available [here](https://ghosteshop.com).

## Development setup

```bash
# Install dependencies
$ yarn install

# Launch dev server
$ yarn dev

# Generate static files
$ yarn generate
```
